# Indecisive Assistant Generator

## Overview
The purpose of this project is to assist indecisive people (aka me) on certain topics, ex. food, outfit, videogames, hobbies, or just a simple ‘yes’ or ‘no’.

## Instructions
1. Ensure you have python downloaded on your computer
2. Open the project folder in VSCode
3. Run the file named 'main.py'

## How the program works
1. When the program starts, the user will see 3 buttons labeled with different categories and when clicked, will lead to different windows (ex. food, TV shows, hobbies). 
2. Next they will be faced with user input where they enter their options into the generator which will be stored into a list.
3. Finally, when all input is entered, the user can click on the generate button where the program will output a random input they entered selected for that category.
    - The data will be stored in the programs data folder if the user want's to check their past picks.
